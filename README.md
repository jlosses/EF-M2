**Proyecto:** Code & Coffee — Landing page de cafetería tech

**Descripción**
- **Propósito:** Página de aterrizaje moderna para una cafetería tech ficticia.
- **Objetivo:** Mostrar ambiente, bebidas y snacks destacados, y facilitar contacto/reserva.

**Características implementadas**
- **Navbar:** Barra con logo y enlaces a secciones (`Inicio`, `Nosotros`, `Menú`, `Galería`, `Contacto`).
- **Hero:** Imagen grande, nombre del restaurante y botones `Ver Menú` / `Reservar`.
- **Sobre Nosotros:** Breve historia y valores.
- **Menú Destacado:** 4 bebidas y snacks tech con imagen, descripción y precio.
- **Galería:** Carrusel de Bootstrap con imágenes y captions.
- **Formulario de Reservas:** Campos mínimos (Nombre, Email, Fecha, Número de personas, Teléfono opcional, Mensaje).
- **Modal:** Confirmación de interés al hacer clic en el corazón de un producto.
- **Footer:** Horario, dirección, teléfono y enlaces a redes.

**Estructura de archivos**
- `index.html` — página principal (en la raíz).
- `assets/css/styles.css` — estilos mínimos complementarios a Bootstrap.
- `assets/js/main.js` — JavaScript reducido (validación básica + modal + fecha mínima).
- `assets/img/` — carpeta para imágenes (actualmente se usan imágenes generadas por Gemini).

**Tecnologías**
- **Bootstrap 5** (CDN) — componentes y layout.
- **Font Awesome** (CDN) — iconos.
- **HTML5 / CSS3 / JavaScript (vanilla)** — código mínimo y maximizando uso de bootstrap.
